# 🔢 Java Number Programs Collection

## 📌 Project Overview

This repository contains a **collection of basic to intermediate Java number programs** designed to practice fundamental programming concepts such as loops, conditions, mathematical logic, and problem-solving.

These programs are helpful for **beginners learning Java**, **students preparing for coding interviews**, and **developers who want to strengthen logical thinking skills**.

---

## 🧠 Concepts Covered

* Java Basics
* Conditional Statements (`if`, `if-else`)
* Loops (`for`, `while`)
* Mathematical Operations
* Number Logic
* User Input using `Scanner`

---

## 📂 Programs Included

### 🔹 Number Property Programs

* Armstrong Number
* Automorphic Number
* Happy Number
* Spy Number
* Strong Number
* Perfect Number
* Palindrome Number

### 🔹 Mathematical Operations

* Factorial Number
* Power of a Number
* Sum of Natural Numbers
* Factors of a Given Number
* Count of Factors

### 🔹 Prime Number Programs

* Prime Number Check
* Largest Prime Number
* Prime Numbers in Given Range
* Prime Numbers (Using Methods)
* Prime Numbers from Ending to Starting

### 🔹 Conversion Programs

* Binary to Decimal
* Decimal to Binary

### 🔹 Series Programs

* Fibonacci Series

### 🔹 Other Programs

* Reverse the Given Number
* LCM (Least Common Multiple)
* GCD / Greatest Common Number
* Leap Year Check
* Sum of Largest and Smallest Prime
* Natural Number Programs
